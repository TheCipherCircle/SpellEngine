## Adaptability Checklist

When evaluating new assets, check for:

- [ ] Separate layers (can isolate parts)
- [ ] Neutral color palette (easy to recolor)
- [ ] Source files included (PSD, AI, etc.)
- [ ] Multiple variations provided (proves theming works)
- [ ] Grid-aligned / snap-ready (easy to combine)
- [ ] Clean topology (won't break when modified)
- [ ] Documentation / examples (shows intended use)
