# Storysmith

**The Narrative Engine for PatternForge**

Storysmith transforms PatternForge tools and security knowledge into immersive interactive adventures.

## Status

**CLOSED SOURCE - PROPRIETARY**

This repository contains premium adventure content and the game engine. It is not open source.

## Overview

Storysmith is the narrative layer that sits atop PatternForge, turning technical security concepts into engaging gameplay experiences through:

- Interactive campaigns with branching narratives
- Pygame-based adventure engine
- Rich lore and world-building content

## Dependencies

- [PatternForge](https://github.com/hashtopia/patternforge) (required)
- Python 3.10+
- Pygame

## Structure

```
storysmith/
├── engine/      # Pygame game engine
└── campaigns/   # Campaign content and scripts

assets/
├── images/      # Visual assets
└── audio/       # Sound and music

lore/            # Narrative documentation
```
